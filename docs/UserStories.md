# EduTrack — User Stories

**Project:** EduTrack Academic Management Platform  
**Team:** VTU Internship 2026 — Team 15

---

## Admin User Stories

### Authentication
- As an admin, I want to log in with my email and password so that I can access the admin portal.
- As an admin, I want to reset my password using OTP sent to my email so that I can recover access if I forget my password.
- As an admin, I want to change my password from the settings page so that I can keep my account secure.
- As an admin, I want to log out of the system so that my session is ended securely.

### Student Management
- As an admin, I want to view a list of all registered students so that I can manage them.
- As an admin, I want to add a new student with their details so that they can access the platform.
- As an admin, I want to edit a student's name and phone number so that their profile stays up to date.
- As an admin, I want to delete a student so that inactive accounts are removed.
- As an admin, I want to search students by name, email, or department so that I can find them quickly.
- As an admin, I want to update a student's status (Active/Suspended/Graduated) so that I can manage their access.
- As an admin, I want to reset a student's password so that I can help them if they are locked out.
- As an admin, I want to bulk import students via CSV so that I can add multiple students at once.

### Teacher Management
- As an admin, I want to view a list of all teachers so that I can manage them.
- As an admin, I want to add a new teacher with their details so that they can access the platform.
- As an admin, I want to edit a teacher's name, phone, and employee ID so that their profile stays current.
- As an admin, I want to delete a teacher so that inactive accounts are removed.
- As an admin, I want to update a teacher's status (Active/Suspended) so that I can manage their access.
- As an admin, I want to reset a teacher's password so that I can help them if they are locked out.
- As an admin, I want to bulk import teachers via CSV so that I can add multiple teachers at once.

### System Settings
- As an admin, I want to enable maintenance mode so that I can prevent student and teacher logins during maintenance.
- As an admin, I want to disable student self-registration so that only admin-added students can join.
- As an admin, I want to set the active academic year and semester so that the platform reflects the current term.

## Teacher User Stories

### Authentication
- As a teacher, I want to log in with my email and password so that I can access the teacher portal.
- As a teacher, I want to reset my password using OTP so that I can recover access if I forget it.
- As a teacher, I want to change my password from settings so that I can keep my account secure.

### Course Management
- As a teacher, I want to view all courses assigned to me so that I can manage them.
- As a teacher, I want to create a new course so that students can enroll in it.
- As a teacher, I want to update course details so that the information stays accurate.
- As a teacher, I want to delete a course so that outdated courses are removed.

### Attendance Management
- As a teacher, I want to mark attendance for students in my course so that their attendance is recorded.
- As a teacher, I want to view attendance records by course and date so that I can track student presence.
- As a teacher, I want to update an attendance record so that I can correct mistakes.
- As a teacher, I want to view attendance statistics per student so that I can identify students with low attendance.

### Marks Management
- As a teacher, I want to enter marks for students in my course so that their performance is recorded.
- As a teacher, I want grades to be auto-calculated based on marks so that I don't have to assign them manually.
- As a teacher, I want to update marks so that I can correct errors.
- As a teacher, I want to view all marks for a course so that I can review student performance.

### Assignment Management
- As a teacher, I want to create assignments for my course so that students know what to submit.
- As a teacher, I want to set a due date for assignments so that students have a deadline.
- As a teacher, I want to update or delete assignments so that I can manage them.
- As a teacher, I want to view student submissions so that I can review their work.

### Course Materials
- As a teacher, I want to upload course materials so that students can access study resources.
- As a teacher, I want to delete course materials so that outdated resources are removed.

### Profile & Settings
- As a teacher, I want to update my name and phone number so that my profile stays current.

## Student User Stories

### Authentication
- As a student, I want to register with my details so that I can create an account.
- As a student, I want to log in with my email and password so that I can access the student portal.
- As a student, I want to reset my password using OTP so that I can recover access if I forget it.
- As a student, I want to change my password from settings so that I can keep my account secure.
- As a student, I want to see an error if I try to log in during maintenance mode so that I know the platform is unavailable.

### Dashboard
- As a student, I want to see a summary of my attendance, marks, and upcoming assignments on the dashboard so that I have a quick overview.

### Courses
- As a student, I want to view all courses in my department so that I know what is available.

### Attendance
- As a student, I want to view my attendance for each course so that I can track my presence.
- As a student, I want to see my attendance percentage so that I know if I am at risk.

### Marks
- As a student, I want to view my marks for each course so that I know my performance.
- As a student, I want to see my grade for each course so that I know my standing.
- As a student, I want to see my CGPA so that I have an overall performance indicator.

### Assignments
- As a student, I want to view all assignments for my courses so that I know what to submit.
- As a student, I want to see assignment due dates so that I can plan my work.
- As a student, I want to submit an assignment so that my work is recorded.

### Course Materials
- As a student, I want to view and download course materials uploaded by my teacher so that I can study.

### Profile & Settings
- As a student, I want to update my name and phone number so that my profile stays current.
- As a student, I want to view my USN, department, year, and semester on my profile so that I can verify my details.
